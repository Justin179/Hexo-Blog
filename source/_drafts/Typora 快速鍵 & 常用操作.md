| - space    | 點清單                    |      |
| 1.space    | 數字清單                  |      |
| tab        | 縮排                      |      |
| shift tab  | 反縮排                    |      |
| ``` enter  | 插入程式碼區塊            |      |
| ``         | 單行 一小段 程式碼        |      |
| --- enter  | 插入YAML Front Matter區塊 |      |
| ctrl t     | 插入表格                  |      |
